# Engineering Rules Learned During M5.5

The following recurring issues were identified during stabilization and should be treated as permanent engineering rules.

### Shopify Placeholders

Use only officially supported placeholder names with `placeholder_svg_tag`.

### Theme Registration

Always verify section registration after extracting or modularizing sections.

### Schema Synchronization

Ensure JSON templates and section block types remain synchronized.

### CSS Scope

Avoid generic selectors that may conflict with Dawn.

### Theme Editor Validation

Every merchant setting must be validated in the Theme Editor before closing a ticket.

### Regression Testing

Every completed ticket must be followed by regression testing on:

- Desktop
- Tablet
- Mobile
- Theme Editor
- Storefront
